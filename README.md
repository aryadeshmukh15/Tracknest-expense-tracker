TrackNest – Expense Tracker Application
A full-stack expense tracking web application that helps users manage daily expenses, categorize spending, and monitor financial habits through an intuitive dashboard.

---
🚀 Features
* Add, update, and delete expenses
* Categorize expenses for better tracking
* RESTful API integration
* Responsive and clean user interface
* Backend validation & exception handling
* Persistent data storage using database

---
🛠 Tech Stack

### Backend
* Java
* Spring Boot
* Spring Data JPA
* REST APIs
* Maven

### Frontend
* React.js
* JavaScript
* HTML5
* CSS3

### Database
* MySQL

### Tools & Technologies
* IntelliJ IDEA
* Git & GitHub
* Postman (API Testing)

---

## 📂 Project Architecture

Client (React) → REST API → Spring Boot Backend → Database
* Controller Layer – Handles API requests
* Service Layer – Business logic implementation
* Repository Layer – Database interaction using JPA
